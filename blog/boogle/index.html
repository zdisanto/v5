This is a brief review of Introduction to HTML and Introduction to CSS. In this review, you will learn that once you have a firm grasp of the basics, you will be able to look at many websites on the internet, such as Google, and be able to create a similar copy of it from scratch.

Part One: HTML

Go ahead and open up your text editor of choice.

Create a folder in a preferable location on your computer and call it Boogle. In that folder, create a file called index.html .

Side Note: If you are using a web browser such as JS Bin or Replit, simply locate the index.html file.

Create a simple index.html boilerplate.

Hint: If you don't remember how to create a HTML boilerplate, check out the Introduction to HTML blog post.

Under the opening <body> tag, create a <h1> tag and write inside it, Boogle.

After the <h1> tag, add a <form> tag and inside of it, create an <input /> tag. When you have finished this, you should have something that looks like the code below:

                    <!DOCTYPE html>
                    <html lang="en">
                        <head>
                            <meta charset="utf-8">
                            <meta name="viewport" content="width=device-width">
                            <title>Boogle</title>
                            <link href="style.css" rel="stylesheet" type="text/css" />
                        </head>

                        <!-- WHERE YOUR CODE STARTS: -->
                        <body>
                            <h1><span>Boo</span>gle</h1> 
                            <form>
                                <input />  
                            </form> 
                            <script src="script.js"></script>
                        </body>
                        </html>
Fun Fact: Google doesn't use an <input> tag, instead they use a <textarea> and a whole lot of <div>'s!

Below the <input> tag, add two buttons. Inside the left button, type "Search", and inside the right button, write "I'm Feeling ____" replace the underscores with an adjective.

An adjective is a word that describes a noun. For example, you can write Sleepy, Silly, Happy, or the Google default Lucky.
